RPA Extração SAP (bases_raw)

SAP GUIExtração

Conecta ao SAP GUI, abre a transação ZCO004 e extrai os relatórios de Real e Compromisso para todas as ordens.

Arquivo

RPA_bases_raw.ipynb

O que faz

  • Lê dim_site.xlsx para obter a lista de ordens SAP
  • Abre nova sessão SAP via COM (win32com)
  • Navega até ZCO004, cola as ordens, executa
  • Exporta resultado para Excel via clipboard
  • Salva EXPORT_REAL.XLSX e EXPORT_COMPROMIS.XLSX na pasta data/raw/

Pré-requisitos

⚠️
SAP GUI precisa estar aberto e logado com pelo menos 1 sessão ativa antes de rodar.

Planilha / Caminho

RecursoCaminho
dim_site.xlsx (entrada)...\PROJETO_PLATAFORMA_FINANCEIRA\data\dimensions\
EXPORT_REAL.XLSX (saída)...\PROJETO_PLATAFORMA_FINANCEIRA\data\raw\
EXPORT_COMPROMIS.XLSX (saída)...\PROJETO_PLATAFORMA_FINANCEIRA\data\raw\

Frequência

1x por dia (manhã), antes do pipeline financeiro.

PipelineGoogle Sheets

Processa os exports do SAP e envia dados consolidados para o Google Sheets (dashboard).

Arquivo

RPA RC SAP (ME51N)

SAP GUICompras

Cria Requisições de Compra automaticamente no SAP via transação ME51N.

Arquivo

RPA_RC_SAP.ipynb

O que faz

  • Lê a planilha base (aba RC ou template dedicado)
  • Para cada linha pendente, abre ME51N no SAP
  • Preenche: material, quantidade, centro, grupo de compras, fornecedor
  • Salva a RC e grava o número gerado de volta na planilha
  • Linhas com status "Feito pela automação" são puladas

Pré-requisitos

⚠️
SAP GUI aberto e logado. Planilha Excel aberta na aba correta.

Planilha

Usa a BASE CONTROLE DE PAGAMENTOS.xlsx — aba "RC" ou template em C:\AUTOMACAO_RC_SAP\

Seção 18 · Referência

Glossário

SAP GUI / COM Object

Interface gráfica do SAP controlada via pywin32. Simula cliques e preenchimento.

ZCO004

Transação SAP para consulta de custos por ordem. Usada pelo RPA_bases_raw.

ME51N / ME21N

Transações SAP: Requisição de Compra (ME51N) e Pedido de Compra (ME21N).

Zendesk / Portal de Serviços

Sistema de chamados (portaldeservicos.pmenos.com.br). RPAs N1/N2 abrem tickets via Selenium.

N1 / N2

N1 = Regularização de NF (campos contábeis). N2 = Lançamento de NF (pagamento direto).

dim_site.xlsx

Tabela dimensão: mapeia ordens SAP → sites/lojas. Usada pelo pipeline.

xlwings

Biblioteca Python que controla Excel desktop via COM. Lê/grava células em tempo real.

MIRO

Número de documento de pagamento SAP. Padrão 51XXXXXXXXX.

Selenium / ChromeDriver

Controla o navegador automaticamente. Usado nos RPAs Zendesk.

PyMuPDF (fitz)

Biblioteca para leitura de PDFs. Usada pelos leitores Lumicenter e Persun.
